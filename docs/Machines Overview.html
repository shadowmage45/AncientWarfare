Machines Overview
Editor's Note (2023): You are reading an archive from 2014. This wiki concerns the Minecraft mod Ancient Warfare, version 1. Neither this wiki nor this version of the mod are being updated. If you are looking for the sequel, Ancient Warfare 2, visit CurseForge. If you are looking for the indie game series, visit JNI Games. If you are looking for Ancient Warfare the magazine, visit Karwansaray Publishers.

Ancient Warfare Machines

Ancient Warfare provides various Machines to help keep your kingdom functioning in top shape. Machines provide means to accomplish tasks that the Civics system alone does not accommodate, and provides alternative means to accomplish some tasks.

Machines

Trash Disposal

The Trash Disposal is a basic block with a 9 slot inventory. When given a redstone signal, it will destroy 1 item (not stack) per tick from its inventory. This block is meant to aid in automation systems where you might have extra unneeded components being produced (seeds, saplings, etc).

Personal Mailbox

The Personal Mailbox provides a means to transport items long distances without the use of couriers. It is, at the core, an item teleporter -- however, it does not teleport items instantly -- rather the travel time is based upon the distance between the source and destination. Each Personal Mailbox must be set to a mailbox name -- this is the name others will select to send things to you at this mailbox. The Personal Mailbox also has slots to pre-set up to 5 destinations, with a corresponding 4-slot inventory for each destination. Every few seconds these output slots will be polled and any items in them packed up and sent towards their destination. Destination slots are labeled as sides (front, top, back, left, right, bottom), with each inventory being accessible from the listed side for tubes/pipes/automation. The larger inventory on the bottom (accessible via bottom side) is for packages sent and delivered to this mailbox. Mailbox inventories and in-route deliveries are stored apart from the blocks that reference them -- much like ender-chests -- you may break a mailbox block and replace it elsewhere, setting the new mailbox to the old name will pull up the old inventory. Each Mailbox name may only be used to name 1 block -- each block must have a unique name.

Industrial Mailbox

The Industrial Mailbox functions exactly like the Personal Mailbox only it has a larger inventory and a single destination slot and inventory. This effectively allows it to send more items in a given period of time, but only to a single destination.

Spot Chunkloader

The Spot Chunkloader will force-load a single chunk, and keep that chunk loaded whether or not any players are nearby.

Deluxe Chunkloader

The Deluxe Chunkloader is an advanced and configurable version of the Spot Chunkloader. It offers a configuration GUI where you may select the specific chunks you want to stay loaded, up to a 5-chunk radius around the chunk-loader block. (11x11 chunk maximum / 121 chunks).

Mechanical Worker

Part of the BuildCraft Plugin, the Mechanical Worker provides a means to convert BuildCraft MJ into Ancient Warfare work units (allows Civics and Crafting to be powered by BC MJ).

Hand Cranked Engine

Part of the BuildCraft Plugin, the Hand Cranked Engine provides a means to power BuildCraft compatible machines using Ancient Warfare worker NPCs.

Food Processor

The Food Processor will turn any input food items into food rations suitable for use with NPCs. The top slots are input, the bottom slot is output. Input is accessible from block top and block sides, output is accessible from block bottom. Requires a redstone 'on' signal in order to function.

Gate Lock

The gate lock provides an ability to lock an Ancient Warfare gate using redstone. Merely place the gate block at one of the lowest corners of the gate, with the keyhole facing the direction you want the lock to activate from (they only accept redstone input on the keyhole side), and provide some means to provide redstone signal (torch, lever, wire, etc). When powered, the lock block will prevent the gate from activating through either redstone or manual interaction.
